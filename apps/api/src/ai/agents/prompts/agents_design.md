I am trying to build an app that will generate a full stack app based on the user's input.
I will use multiple agents to build the app.

The first agent will be the master agent.
The master agent will be responsible for the overall app design and the overall app structure and requirements.

Then it will pass the requirements to the ui agent.
The ui agent will be responsible for the overall pages descriptions and sections descriptions.

Each section will be passed to the uiSection agent.
The uiSection agent will be responsible for the overall section design and the different components that will be used in the section.

Each component will be passed to the uiComponent agent.
The uiComponent agent will be responsible for the overall component design and requirements,props, states and dynamic data.
